🛡️ Bharti AXA Life Insurance Claims Analytics Dashboard

An interactive Power BI dashboard designed to analyze life insurance policies, premium collections, claim performance, customer demographics, and risk indicators. The dashboard helps insurance companies monitor claim trends, evaluate policy performance, and make data-driven business decisions.

1. Short Description / Purpose
The Bharti AXA Life Insurance Dashboard provides a comprehensive view of premium collections, claim amounts, coverage values, and customer behavior. It enables stakeholders to track claim performance, identify risk patterns, and evaluate policy profitability through interactive visualizations and KPIs.

2. Tech Stack
The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Dashboard development and visualization.
🐍 Python – Data cleaning, preprocessing, and validation.
🗄️ SQL Server – Data storage and querying.
❄️ Snowflake – Cloud data warehousing and analytics.
📑 Excel – Initial data exploration and quality checks.
🔄 ETL Process – Data extraction, transformation, and loading.
📈 DAX (Data Analysis Expressions) – KPI calculations and business metrics.
🌐 GitHub – Version control and project documentation.

5. Dashboard Insights
Customer Analysis
Gender-wise customer distribution.
Nearly equal distribution between male and female policyholders.
Claim Status Analysis
Rejected claims represent the highest share.
Pending and settled claims contribute equally.
Policy Type Analysis
Health policies generate the highest premium contribution.
Auto and Travel policies show similar premium performance.
Risk Metrics
Loss Ratio and Claim Ratio help identify high-risk policy segments.
Claim Frequency measures claim occurrence trends.
